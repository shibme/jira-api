A simple JIRA API Client

### Maven Dependency
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
    <artifactId>jira-api</artifactId>
    <version>0.1.1</version>
</dependency>
```
